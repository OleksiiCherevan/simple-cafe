# Simple Cafe

## This is my practice in scss and react. This application has only desktop design at the moment. 

## To run this application you should:
* git clone https://github.com/OleksiiCherevan/simple-cafe/
* cd simple-cafe

#### if you use yarn 
* yarn
* yarn start

#### if you use npm
* npm install
* npm run start
